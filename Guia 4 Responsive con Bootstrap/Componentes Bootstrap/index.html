Mensajes de Alerta

La sintaxis es:
<div class="alert alert-success">
<strong>Success!</strong> Indicates a successful or positive action.
</div>

Mensajes de Alerta tradicionales

Exito! Indica un proceso exitoso.
Informacion! Inidica una informacion neutral de cambio o accion.
Advertencia! Indica una advertencia que puede necesitar atención.
Peligro! Indica un peligro o una acción potencialmente negativa.

Barras de Progreso

La sintaxis es:
<div class="progress">
<div class="progress-bar" role="progressbar" aria-valuenow="70"
aria-valuemin="0" aria-valuemax="100" style="width:70%">
<span class="sr-only">70% Complete</span>
</div>
</div>

Se les puede agregar texto o en su defecto, el porcentaje actual en la barra

70%

Se les puede agregar rayas y color con una escala similar a la de los mensajes de alerta, además de animarlos o no

40% Completado (Exito)
50% Completado (Informacion)
60% Completado (Advertencia)
70% Completado (Peligro)

Tambien se puede crear una barra segmentada que represente las distintas partes de un todo
